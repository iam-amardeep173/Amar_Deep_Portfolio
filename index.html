Sign Language Recognition Project

Duration: August 2022 - November 2022

Objective: Develop a computer application to recognize hand gestures of Indian Sign Language and display the corresponding text.

Technologies Used:

  • Programming Languages: Python
  • Libraries/Frameworks: PyTorch, OpenCV
  • Development Environments: VSCode, Google Colab

Models Used: LSTM and Transformer (Transformer achieved 94.64% accuracy)

Approach:

  • Data collection and preprocessing using OpenCV for hand gesture detection.
  • Model training using PyTorch to build LSTM and Transformer models.
  • Evaluation and comparison of model performance, with the Transformer model significantly outperforming the LSTM model.
